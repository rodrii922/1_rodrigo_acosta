Ejemplo de Etiquetas HTML

Formato de texto

Este documento muestra varias etiquetas HTML.

Tipos de etiquetas

  • <b>: texto en negrita.
  • <em>: texto enfatizado.
  • <i>: texto en cursiva.
  • <u>: texto subrayado.
  • <strong>: texto importante.
  • <small>: texto pequeño.
  • <sub>: texto subíndice.
  • <sup>: texto superíndice.
  • <ins>: texto insertado.
  • <del>: texto borrado.
  • <mark>: texto marcado.
